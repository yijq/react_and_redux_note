# React 起步

## 环境配置
- 需要nodejs环境
- react依赖于庞大的技术栈,babel转义,webpack打包等等,暂时可以不用深入研究
- 可以直接用FB官方的工具,create-react-app:
    ```javascript
    npm install -g create-react-app 
    create-react-app first_react_app
    
    ```
- react语法为jsx,而且现在都使用ES6来编写
- react工作方式:
    - UI = render(data)
    - Virtual DOM
    - Diff算法
